# Session 7 Learning Logs

In <head>:
- UTF-8: Universal language, every computer can render 
- viewport: makes sure there is responsiveness on the page; related to how you define the width of your device 


<br> creates a line break within text. The sentences belong to the same paragraph but the second sentence shifts down to the next line, creating a line break

<hr> creates a horizontal line

<strong> used for important content (makes content bold)

<b> also used for making text bold (visual only)

<em> used for emphasis (makes text italic)

<i> also makes text italic (visual only)

<u> adds underline to text

<span> is for inline styling of text. example- color, highlight, background color for particular text

<div> is a container for grouping content 

<footer> typically contains copyright or site information 

<li> is used for lists; can also have sub-lists

Shapes example: Rectangle is <rec>
