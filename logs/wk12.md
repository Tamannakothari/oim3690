# Week 12 learning Logs

Images
- made of pixels
    - Every digital image is a grid of tiny colored squares called pixels
    - A 1920x1080 image has over 2 million pixels
    - Each pixel has a color value
- array of numbers

Example colors:
1. pure red
2. pure green
3. white (all colors at max)
4. black (no colors)
5. grey (equal amounts)
