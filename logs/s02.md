# Session 2 Learning Logs

Version Control: used to track of changes to files over time, instead of making separate files whenever anything is changed or updated

Git: most widely used version control software
- better to use online provider
- major providers: GitHub, GitLab, BitBucket

# Advantages of GitHub
- Version control
- Backup
- Portfolio
- Collaboration
- Deployment 

# Steps:
- New repository 
- Name repository
- Add description
- Choose location for repository 
- Check "Initialize this repository with a README"
- Choose Git ignore type
- Create repository 
- Commit changes
- Publish repository 

# Basic Workflow 
- Pull: latest changes 
- Edit files in VS Code
- Pull: before committing 
- Commit: save changes locally
- Push: upload changes to GitHub
