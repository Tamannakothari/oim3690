# Week 5 Learning Logs

# Review
3 ways to add CSS to a page:
- Inline (avoid)
- External (recommended)
- Internal (in <head>)

Cascading: When multiple styles conflict, CSS follows this priority:
- Inline styles (highest priority)
- Internal/External stylesheets (last one wins)
- Browser defaults (lowest priority)

# CSS Syntax
selector {
    property: value;
    property: value;
}

# Basic Selectors, Targets & Examples
- Element     All elements of type     p {}
- Class       Elements with class      .intro {}
- ID          Element with ID          #header {}
- Universal   All elements             * {}


- Adding borders: helps to see how much space exactly each element takes
- Then to remove gaps- margin:0

- Class selector: Classes can be used on multiple elements
- ID selector: IDs should be unique (only one per page)

# Selector Specificity
From lowest to highest priority:
1. Element selectors
2. Class selectors
3. ID selectors
4. Inline styles

# Every Element is a Box
- Content- actual content
- Padding- space inside border
- Border- edge
- Margin- space outside border
