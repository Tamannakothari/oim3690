# Week 10 Learning Logs 


# Function 
A reusable block of code that performs a task

// Define function
function greet() {
  console.log("Hello!");
}

// Call function
greet();  // prints "Hello!"
greet();  // prints "Hello!" again


# Parameters
they let functions work with different inputs each time you call them

# Return Values 
Use return to send a result back. Without it, the function returns undefined.

# if...else if...else 
Conditions are checked in order. The first true condition runs and the rest are skipped.
