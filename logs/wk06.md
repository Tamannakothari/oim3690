Most common ways to center a div:
- Flexbox

# Flexbox Basics
- parent div, children inside

.container {
    display: flex;
}
This makes all direct children flex items that
- line up in a row by default 
- shrink/grow to fit
- can be easily aligned


- space-between: if you multiple items and want to space them out without measuring, it takes care of that
- perfect centering: easiest way to center anything
